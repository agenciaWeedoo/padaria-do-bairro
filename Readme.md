DOCUMENTAÇÃO DO PROJETO: PADARIA DO BAIRRO
============================================================
Desenvolvedor: Carlos Henrique Lopes Macedo (Agência Weedoo)
Data: Maio de 2026
Versão: 2.0 (Módulo de Interatividade)

1. OBJETIVO DO PROJETO:
Desenvolvimento de um ecossistema web para uma padaria artesanal, 
focando em experiência do usuário (UX), acessibilidade (a11y) e 
comunicação direta através de formulários inteligentes.

2. ATIVIDADES REALIZADAS:
- ATIVIDADE 1: Estrutura de dados com tabelas semânticas (Cardápio e Horários) 
  utilizando tags de hierarquia (thead, tbody, tfoot).
- ATIVIDADE 2: Curadoria de mídia com galeria responsiva e vídeo institucional 
  com controles nativos e descrições alternativas (alt text).
- ATIVIDADE 3: Padronização de arquitetura de arquivos (Arquitetura Limpa):
  /css (estilos), /images (assets visuais), /videos (mídia).
- ATIVIDADE 4 (PRÁTICA 3): Implementação de 4 módulos de interação:
    * Contato Geral: Validação de campos e máscaras de telefone.
    * Encomendas Especiais: Sistema de seleção de produtos e datas.
    * Feedback do Cliente: Sistema de avaliação por rádio e permissões.
    * Clube de Vantagens: Cadastro completo com dados de endereço e newsletter.

3. DIFERENCIAIS TÉCNICOS:
- Design Surrealista: Uso de gradientes de malha (mesh gradients) e 
  animações de aurora no CSS3.
- UX Avançada: Feedback visual nos formulários através de seletores 
  de estado (:focus, :hover).
- Validação Nativa: Uso estratégico de tipos de input (email, date, tel) 
  e atributos de validação (required, pattern).

4. TECNOLOGIAS UTILIZADAS:
- HTML5 Semântico e Avançado
- CSS3 (Flexbox, Keyframe Animations, Glassmorphism)
- Controle de Versão: Git & GitHub (Agência Weedoo)
