DOCUMENTAÇÃO DO PROJETO: PADARIA DO BAIRRO
============================================================
Desenvolvedor: Carlos Henrique Lopes Macedo (Agência Weedoo)
Data: Maio de 2026

1. OBJETIVO DO PROJETO:
Desenvolvimento de uma landing page institucional para uma padaria artesanal, 
focada em acessibilidade, organização semântica e design temático.

2. ATIVIDADES REALIZADAS:
- ATIVIDADE 1: Implementação de tabelas semânticas para cardápio (10 produtos) 
  e horários de funcionamento, utilizando thead, tbody e tfoot.
- ATIVIDADE 2: Galeria de mídia com mais de 5 imagens e vídeo institucional 
  centralizado. Uso rigoroso de atributos 'alt' para acessibilidade.
- ATIVIDADE 3: Reestruturação do diretório de arquivos para o padrão:
  /css (estilos), /images (produtos, ambiente, logo), /videos (mídia).

3. TECNOLOGIAS UTILIZADAS:
- HTML5 Semântico
- CSS3 (Gradientes, Flexbox)
- Estrutura de pastas padronizada para manutenção simplificada.
